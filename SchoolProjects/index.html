There Will Come Soft Rains

This game was created using a modded version of a programing language called scratch. Scatch is a relatively simple block-coding language.
I created this game based on the book There will come soft rains by Ray Bradbury.
The core gameplay is based on keeping the house clean. This is based on the fact that the book repetitively talked about how the house kept itself clean
The game also has fire, another aspect of the story.
The end goal of the game was to inclue stuff like the breakfast that you would have to clean up, but I didn't have enough time.
With more time I would have also added a final boss where a bunch of fire would spawn.
